# web_xml의 개요, 기능, 활용
## Web.xml개요
#### web.xml이란?
  + Web Application의 Deployment Descriptor(환경파일 : 배포서술자, DD파일)로서 XML 형식의 파일
  + 모든 Web application은 반드시 하나의 web.xml 파일을 가져야 함
  + 위치 : WEB-INF 폴더 아래
  + web.xml 파일의 설정들은 Web Application 시작시 메모리에 로딩됨. (수정을 할 경우 web application을 재시작 해야함.)
